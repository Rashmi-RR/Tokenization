Who Made This?

This project was created by a web development enthusiast eager to learn and apply Git in a practical way. The purpose was to build a simple yet meaningful site that could be version-controlled and deployed using GitHub. HTML, CSS, and Git were the primary tools used in this exercise.

The developer believes in hands-on learning. Instead of just reading about Git or HTML, they chose to build this small portfolio project and track every file, commit, and branch to better understand the workflow used in professional development environments.

Project Goals

The goals of MyProject include practicing Git operations like branching, committing, merging, and pushing code to a remote repository. It also aims to create a clean and readable HTML structure that can be used as a reference or template in future web projects.

In the future, the project could evolve into something more dynamic using JavaScript or integrate with a backend framework like Flask or Node.js. But for now, it stands as a strong example of a static site managed with Git and deployed on GitHub Pages or similar platforms.
